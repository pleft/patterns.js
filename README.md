#patterns.js

Javascript Patterns with examples

##patterns covered:

    - constructor pattern
    - decorator pattern
    - module pattern
    - prototypal pattern
    - singleton pattern
    - factory pattern
    - facade pattern
    - observer pattern
    - surrogate pattern (not sure if it is *pattern*)
